# php_curl

#### 项目介绍
php简单封装curl

#### 环境要求
- 必须有php_curl扩展

#### 文件说明
- BaseHttp.php:http基类
- HttpClient.php: 接口,定义所有的HTTP操作,设置函数
- HttpGet.php: HTTP的GET实现类
- HttpPost.php: HTTP的Post实现类
- Config.php: HTTP的配置文件
- test.php: 测试文件

#### 使用说明

1. 导入BaseHttp.php,HttpClient.php,HttpGet.php,HttpPost.php文件即可使用
